## ЁЯУМ About

**TinDog** is a playful front-end project that mimics a landing page for a "Tinder for Dogs" app. It was built as a hands-on exercise during Angela Yu's Web Development Bootcamp on Udemy, focusing on responsive web design, Bootstrap components, and modern CSS layout techniques.
